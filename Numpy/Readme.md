## ðŸ“– What is NumPy?
**NumPy** (Numerical Python) is the core library for scientific computing in Python. It provides a high-performance multidimensional array object and tools for working with these arrays. 

> **Technical Note:** While NumPy is used via Python, it is primarily **based on C**. This allows it to perform complex mathematical operations at much higher speeds than standard Python lists by utilizing contiguous memory allocation.
